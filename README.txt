◎シークレットプログラムについて
　「シークレットプログラム」（参考プログラム）は、サンプルプログラムとは異なる処理をするプログラムです。実行ファイルと絵素材のみ提供しています。応用力を鍛えるために、シークレットプログラムと同じ挙動をするよう、サンプルプログラムを基にしてカスタマイズしてみましょう。
　シークレットプログラムのフォルダに、ヒントが書かれたテキストファイルも同梱してありますので、参考にしてください。
